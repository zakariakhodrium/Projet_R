# Piccross Game

Welcome to Piccross! Piccross is a puzzle game where you use logical deduction to reveal pictures on a grid. This game offers an interactive experience and hours of fun solving puzzles in three difficultu levels.

## Installation

To install Piccross, you need to have the `devtools` package installed in R. You can install it using:

```R
install.packages("devtools")
devtools::install_github("zakariakhodrium/Projet_R")
PiccrossR::Piccross_R()
```
